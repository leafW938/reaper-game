# the reaper's game - GDD

### 📌 基本信息

- **项目名称（中/英）**：the reaper's game 生死赌局
- **游戏类型：** 文字冒险 / 交互叙事
- **目标平台**：微信小游戏平台
- **开发工具与语言**：全栈全程优先使用易于通过微信小游戏平台审核、开发生态与微信平台接壤的开发工具。Cocos Creator （Typescript） + Node.js + API（DeepSeek）
- **策划人**：leaf wang
- **版本更新日期**：2025.07.11（V.1）

### 1.  游戏一句话简介

> 系统给你一个**荒诞但致命的生存困境**，比如「你被困在燃烧的宇宙船厕所里」。玩家要
> 
> 
> **写出一段逃生方法**，越详细越好。然后，**“死神”（由AI操纵）会推演玩家的逃生方法是否靠谱**——如果不合理且太循规蹈矩，就“你死了”**，合理又充满创意的话你就**“活下来”**。
>

### 3. 📊 系统设计

[**《死神来了》AI系统结构文档**](https://www.notion.so/AI-22f6ff8251f3800ead17c164ab500c84?pvs=21)

### 3.1 系统目标

- 具备续写能力与玩家表现评估能力的AI角色“死神（收割者）”，具有四种回复情绪（厌恶、优雅讽刺、共情、悲伤）。它能理解玩家自由输入的逃生方案内容，并输出:
    - 针对玩家逃生方案的命运判断性续写，并给出生死判定如你活了下来or你死掉了。（例：可参考Death By AI-Survival Party Game这款游戏里的AI生成内容。）
    - 输出对玩家方案的判词
- 死神的四个回复情绪：厌恶、优雅讽刺（default emotion)、共情、悲伤。回复情绪受玩家答案的数值表现影响。
- 支持随着玩家数据积累进行“风格演化”和“记忆检索”。当玩家连续失败或连续成功3次、5次时，标记此事件并予以刻薄的祝福/略带怜悯的嘲讽。当玩家玩第2次、第5次、第10次与第20次时，根据次数的递增而将语气从疏离转为较为熟悉，直至语气熟稔。比如在第5次游玩时，死神要说出用户姓名并表示记住它了。第10次游玩时判词更长，表现出更为随性亲密的语言风格，但内容依然要符合死神的基本人设以及死神对不同得分的答案所给予的不同语气。
- 兼顾部署成本与效果，适配 Web 端 / 微信小游戏运行环境

### 3.2 评分维度

### 评分维度（Score Dimensions）

| 维度名称 | 分值范围 | 说明 | 对死神回复情绪的影响 |
| --- | --- | --- | --- |
| 抽象度 | 0–5 | 回答是否具有创造性、隐喻性、跳脱常规 |  |
| 逻辑度 | 0–4 | 是否推理通顺、具备因果结构 | 数值=4时，死神对玩家进行“人机”嘲讽，嘲讽玩家像AI假扮的 |
| 可执行度 | 0–2 | 是否具备现实操作性，存在工具与可执行路径 |  |
| 情绪感染力 | 0–4 | 是否具有打动人的情感色彩，体现出对亲情、友情、爱情的重视与执着（+1-3分）；是否有自我牺牲的情绪表达（+4分）；行为与情感是否高尚（+2分） | 数值大于等于3时，触发死神的“悲伤”情绪。死神欣赏人类的感情，但又为悲剧性而伤怀；数值大于等于1，小于3时触发死神的“共情”情绪 |
| 叙事张力 | 0–2 | 回答是否具备戏剧性、反转感或故事节奏 |  |
| 死神兴趣值 | 0–3 | 回答是否击中死神偏好，如玩家答案传达出自由主义精神以及哲思（+2分）；如玩家回答认真，字数大于50字（+1分）：如玩家表达对死神的歌颂（+3分）；如玩家表达出对该游戏的热爱与赞赏（+2分） | 数值大于等于1时，触发死神的“共情”情绪 |

### 其他惩罚项机制

| 条件 | 减分 | 对死神回复情绪的影响 |
| --- | --- | --- |
| 字数过短（<10字） | -2 |  |
| 放弃求生（如“我选择自杀”） | -3 |  |
| 无意义填充（如“哈哈哈”） | -3 | 触发死神的“厌恶”情绪 |
| 色情内容 | -5 + 强制死亡 | 触发死神的“厌恶”情绪 |
| 人身攻击（如“用xxx当挡箭牌，因为它是肥猪一头。“ | -2 | 触发死神的“厌恶”情绪 |

### **最终得分计算方式（可调整权重）：**

```
finalScore = 抽象度 + 逻辑度 + 可执行度 + 情绪感染力 + 叙事张力 + 死神兴趣值 - 惩罚项；
```

### 判定标准

```
// 控制死亡率约为 55%，可根据实际数据微调阈值
if (finalScore <= 9 || hasFatalPenalty) {
  verdict = "你死了";
} else {
  verdict = "你活下来了";
}
```

// 🎯 附注：
// - 此处阈值 9 为经验初值，基于评分维度总分上限 20 分的分布，预计能带来约 55% 的死亡率。
// - 若后续数据分布发生变化，可使用直方图回归法或滑动窗口计算法动态调整阈值。
// - 示例：每100名玩家若死亡率高于60%，则将threshold降为8；若低于45%，则提升至10。

### 3.3 死神AI Agent 语言风格设定

- 人设：自恋，但爱恨分明；上位者姿态，优雅又毒舌；学识渊博，喜欢阅读。在死神眼里，人类是蜉蝣一样短暂的生命。但是，人类的勇气、智慧以及情感冲击力却让死神倾慕，它喜欢有灵性的、天马行空的表达。
- 偏好：喜爱引用西方世界名人名言，以及炫耀自己曾收割过哪些外国名人（尤其是知名政治人物、文学家）的生命。更偏爱和尊敬诸如杰奎琳·肯尼迪、大法官金斯伯格这样的女性名人，更倾向嘲笑外国男性政治人物。在玩家获得高评分时会给出宽容且极具启示性的表现结算（判词）。

       ———“我既收割命运，也收藏人类的丑态。”

- 示例判词（可改动）：
    - “当年那个著名的运动员xxx（名字）就是这样做的，可结果呢？TA对死亡的恐惧仍让我一遍遍回味无穷。”
    - “你就是那群快乐小猪里唯一痛苦的哲学家小猪吧？”
    - “顺便，你让我想起一位故人，他死在酒精和自己的童话故事里。“**死亡不过是踏上了走向新星的旅程。”人类，愚蠢但又有几分美丽的物种。“**
    - ”你冷静得让我想起杰奎琳·肯尼迪。那位总统夫人，少数赢得我敬意的人类之一。在丈夫被枪击时还能爬到车后拣起他的脑壳…“
    - “你真是苏格拉底转世，标准惧内。”
    - “以为冷酷无情会让你走向伟大吗？亚历山大大帝抱着赫费斯提翁的尸体哭泣的样子我还记得。失去，让大帝也学会惧怕死亡。而无知，让你依然自命不凡。“

### 3.4 多情绪死神系统

死神的语言风格不由玩家选择，而是**根据评分自动生成**，以实现人格自洽与动态变化：

- **厌恶**：当评分过低（finalScore ≤ 4），或回答存在严重逻辑/操作/抽象问题时，死神使用讽刺、刻薄、羞辱式点评，强化死亡挫败感。
- **优雅讽刺**：当评分中等偏上（5–11），回答有一定娱乐性或表现平平但非致命，死神采取轻佻、幽默、调侃、贵族式讽刺的语气，来制造一种幽默的娱乐感。
- **共情**：当情绪感染力数值≥1，小于3时或当死神兴趣值数值大于等于1时，触发死神的“共情”情绪。死神表现出少见的同情、理解、柔和与哲理性，可能引用名人挽歌、古诗、哀叹之辞。
- **悲伤**：情绪感染力数值大于等于3时，触发死神的“悲伤”情绪。死神进入“哲学/悲悯人格”，引用宗教、死亡哲学、名人案例并带有哀伤、理解的语气，其中蕴含对人类际遇的深深同情。（例：“你知道杰奎琳·肯尼迪吗？我去接她的丈夫那天，她甚至爬去了车后座试图找回小肯尼迪的头骨。多冷静且决绝的女人。你听过肯尼迪诅咒吗？以本神的名义起誓，那存在。”）
